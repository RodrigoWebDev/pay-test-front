# React Whater App

## Live demo
[Click here](https://romantic-gates-0ddd40.netlify.com/)

## Build With

* HTML5
* CSS3
* Javascript
* React
* [Weather API](https://openweathermap.org/api)

## How to Install

```
yarn
```

## How to use

Development server:
```
yarn start
```

Build
```
yarn build
```
